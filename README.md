# Nutrigenie Application 

This is a mobile application project which helps user create balanced meals from items in their fridge keeping in mind their dietary complications.



## Features
- Log In/ Sign Up with encrypted password
- Edit dietary complications in account section
- Add/remove items from the fridge section
- Generte balanced meal options from the items in the fridge according to the dietary complications
- Accept a meal option and automatically remove those items from the fridge / Decline meal options  



## Technologies Used
- Java
- Android Studio
- XML
- Gradle
- Android SDK
- SQLite

## Download and Use

# Prerequities 
- Android Studio (latest stable version recommended)
- Android SDK installed
- Java Development Kit (JDK) 17 (or the version required by the project)

# Installation 
1. Clone the reposiroty

```git clone https://github.com/ilincabitir/Nutrigenie```

2. Open Android Studio.

3.  Select Open and choose the cloned project folder.

4. Allow Android Studio to:
  - Sync the Gradle files.
  - Download any missing dependencies.

5.  Connect an Android device with USB debugging enabled or start an Android Emulator.

6.  Click Run to build and launch the application.
